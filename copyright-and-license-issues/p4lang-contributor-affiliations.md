# Introduction

This article records the company where selected p4lang contributors
worked, and over what range of dates.  This is intended to aid those
who in the future wish to review the copyright holders of p4lang
project source files added by those people.

Note that the earliest date of any file added to a p4lang repository
is in 2014, so this history of affiliations has no need to go back
earlier in time than that.

Any time range that says "to present" means "until 2026-Jun" when this
file was last updated.
